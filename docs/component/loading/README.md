# Loading 加载中

### 安装
``` bash
npm install @plutojs/loading
```

### 引用
``` js
import Loading from '@plutojs/loading';
import '@plutojs/loading/build/index.css';
```

### 显示loading
``` js
Loading.show('提交中');
```

### 隐藏loading
``` js
Loading.hide();
```
