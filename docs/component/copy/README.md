# Copy 复制

### 安装
``` bash
npm install plutojs/copy
```

### 引用
``` js
import Copy from '@plutojs/copy';
```

### 判断是否支持复制
``` js
Copy.isSupported();
```

### 复制操作
``` js
Copy.copy(ele: HTMLElement);
```
